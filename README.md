# 🐍 Snake-Eater Game

A classic Snake game built with **JavaScript** and the **Canvas API**.  
Guide the snake to eat food, grow longer, and avoid collisions with walls or itself!

---

## 🎮 Features
- Smooth gameplay using the HTML5 Canvas API
- Snake grows longer each time it eats food
- Score tracking system
- Game-over detection (collision with walls or snake body)
- Simple, clean UI
